Offshore — Waitlist (single-file build)
========================================

Just double-click index.html. That's it.

Everything (HTML, CSS, React components, animations) is in one file so
your browser doesn't need to load anything from disk. You do need an
internet connection the first time — React, GSAP and Google Fonts are
pulled from a CDN.

Note: the waitlist form is client-side only here — submitted emails log
to the browser console (Cmd+Opt+J on Mac, F12 on Windows) but aren't
saved anywhere. To capture real signups, deploy the Next.js project to
Vercel.
